# Real-Time-Speech-Translator
Real-time Tamil to English speech-to-speech translator using SpeechRecognition, Google Gemini API, and text-to-speech synthesis

# Overview
This project implements a real-time Tamil to English speech-to-speech translation system.
It captures Tamil speech through a microphone, converts it to text using speech recognition, translates it into English using Google's Gemini Generative AI model, and speaks the translated output using text-to-speech synthesis.
The system is designed to provide smooth and low-latency cross-language voice interaction.

# Features
- Real-time Tamil speech recognition
- AI-powered translation using Gemini API
- Automatic English speech output
- Low-latency processing
- Prompt-engineered translation control

# Tech Stack

- Python
- SpeechRecognition
- Google Generative AI (Gemini 2.5 Flash)
- pyttsx3 (Text-to-Speech)
- Microphone Input
